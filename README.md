# typescript-interfacez

Ejercicios de primeros pasos con TypeScript: interfaces, interfaces opcionales, propiedades de solo lectura, funciones tipadas y primeras clases. Contiene las secciones 6 y 7 de un curso de TS.

## Contexto

Repositorio de autoestudio (julio 2023) con archivos `.ts` por cada concepto practicado. Cada carpeta corresponde a una sección del curso:

- `interfacez-seccion-6/` — `primerainterfaz.ts`, `interfacesopcional.ts`, `sololecturas.ts`, `funciones.ts`
- `seccion-7/` — introducción a clases (`elrror.ts`)

> Nota: el nombre del repo contiene un typo (`interfacez` → `interfaces`). Pendiente de renombrar en GitHub.

## Tecnologías

- TypeScript
- Node.js (para compilar con `tsc`)

## Ejecución

```bash
npm install -g typescript
tsc nombre-del-archivo.ts
node nombre-del-archivo.js
```
